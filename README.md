# Forum-Hub-Challenge-Back-End

<p align="center">
✅Projeto concluído✅
</p>

O **FórumHub** é uma API REST robusta desenvolvida para simular o funcionamento de um fórum de discussões. O projeto foca no gerenciamento de tópicos, permitindo que usuários autenticados realizem todas as operações de um CRUD (Criar, Consultar, Atualizar e Excluir), com persistência de dados e segurança via tokens JWT.

**🚀 FUNCIONALIDADES**

* **CRUD de Tópicos:** Gerenciamento completo de dúvidas (Título, Mensagem, Autor e Curso).
* **Segurança com JWT:** Autenticação stateless utilizando JSON Web Tokens para proteger os endpoints.
* **Validação de Dados:** Regras de negócio para impedir tópicos duplicados e garantir campos obrigatórios.
* **Paginação e Ordenação:** Listagem otimizada de tópicos para melhor performance e usabilidade.
* **Persistência em Banco de Dados:** Uso de MySQL e Flyway para controle de versões do banco.


**🛠️ TECNOLOGIAS UTILIZADAS**
* **Java 25 / Spring Boot 3:** Framework base para o desenvolvimento da API.
* **Spring Security:** Para controle de autenticação e autorização.
* **Auth0 JWT:** Biblioteca para geração e validação de tokens seguros.
* **Spring Data JPA / Hibernate:** Para mapeamento objeto-relacional e persistência.
* **MySQL:** Banco de dados relacional.
* **Flyway:** Gerenciamento de migrations do banco de dados.
* **Lombok:** Para redução de código boilerplate (getters, setters, construtores).
* **TRELLO:** Ferramenta visual de gestão de projetos e tarefas, baseada no método Kanban. Ele foi utilizado para organização de todas as etapas necessárias para a conclusão do desafio.
